Wokwi 임포트 방법
  1. wokwi.com 접속 → New Project → Arduino Uno
  2. 좌측 파일 목록에서 diagram.json 클릭
  3. 내용 전체를 다운로드한 파일로 교체
  4. 저장 후 시뮬레이션 실행

핀맵 (Arduino Mega 2560)

센서

A0토양수분 센서 1민트
A1토양수분 센서 2레몬밤
A2토양수분 센서 3바질
D2DHT22 온습도10kΩ 풀업저항 필수
D3BH1750 SDA소프트웨어 I2C
D4BH1750 SCL소프트웨어 I2C

액추에이터

D5SSR-2 신호펌프 비상차단
D6SSR-1 신호WO-100D 환풍기
D7DPDT 릴레이 AK-6060 열기
D8DPDT 릴레이 BK-6060 닫기
D9LED 녹색220Ω 직렬
D10리밋 스위치 A측창 열림 감지
D11리밋 스위치 B측창 닫힘 감지
D22전자밸브 1민트 구역
D23전자밸브 2레몬밤 구역
D24전자밸브 3바질 구역

모드 스위칭

D12토글 스위치AUTO↔MANUAL
A3푸시버튼 1수동 제어
A4푸시버튼 2수동 제어
A5푸시버튼 3수동 제어

배선 주의사항

⚠️ DHT22: DATA 핀과 VCC 사이에 반드시 10kΩ 풀업저항 연결
⚠️ SSR-1/SSR-2: 방열판 필수. 없으면 수분 내 과열 손상
⚠️ 관수 중 측창 열기 금지 — 물 튀김 사고 방지
⚠️ DPDT 릴레이 D7/D8: 동시 ON 절대 금지 — 24V 쇼트. 50ms 딜레이 필수
⚠️ 모든 GND는 공통 접지 (Arduino GND)
