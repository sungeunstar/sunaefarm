핀용도최대 전류
5VDHT22, BH1750, 토양수분 센서, 릴레이 VCC, ULN2003 COM500mA (USB 공급 시)
3.3V사용 안 함—
GND모든 부품 공통 접지—
5V 핀 전류 주의
Arduino Uno의 5V 핀은 USB 공급 시 약 500mA까지만 안전하게 사용할 수 있습니다. 환풍팬(12V DC)은 반드시 외부 12V 어댑터로 별도 공급해야 합니다. 팬을 5V Arduino 핀으로 구동하면 Arduino가 리셋되거나 손상됩니다.

DHT22 배선 상세

DHT22 핀 1 (VCC)  ──────────────── Arduino 5V
DHT22 핀 2 (DATA) ──┬─────────── Arduino D2
                    └── 10kΩ ── Arduino 5V  ← 풀업저항
DHT22 핀 3 (NC)   ── (연결 안 함)
DHT22 핀 4 (GND)  ──────────────── Arduino GND
10kΩ 풀업저항 없으면 DHT22가 동작하지 않습니다
DATA 핀과 VCC 사이에 10kΩ 저항을 반드시 연결해야 합니다. 없으면 센서 값이 -1 또는 NaN으로 읽힙니다.

릴레이 모듈 배선 상세

릴레이 모듈 VCC ──── Arduino 5V
릴레이 모듈 GND ──── Arduino GND
릴레이 CH1 IN  ──── Arduino D5  (LOW = 펌프 ON)
릴레이 CH2 IN  ──── Arduino D6  (LOW = 팬 ON)

릴레이 CH1 COM ──── 펌프 전원 +
릴레이 CH1 NO  ──── 펌프 +극   (Normally Open)

릴레이 CH2 COM ──── 12V 어댑터 +
릴레이 CH2 NO  ──── 팬 +극     (Normally Open)
릴레이는 LOW 신호로 켜집니다
일반적인 5V 릴레이 모듈은 IN 핀에 LOW(0V) 신호가 오면 릴레이가 ON됩니다. Arduino 코드에서 digitalWrite(RELAY_PIN, LOW) = 켜기, HIGH = 끄기임을 주의하세요.

ULN2003 + 28BYJ-48 배선

ULN2003 IN1 ──── Arduino D8
ULN2003 IN2 ──── Arduino D9
ULN2003 IN3 ──── Arduino D10
ULN2003 IN4 ──── Arduino D11
ULN2003 COM ──── Arduino 5V
ULN2003 OUT1~4 ─ 28BYJ-48 커넥터 (세트 구매 시 자동 연결)
스테퍼모터 Arduino 직결 금지
28BYJ-48 스테퍼모터를 Arduino 핀에 직접 연결하면 Arduino가 손상됩니다. 반드시 ULN2003 드라이버 보드를 경유해야 합니다.
