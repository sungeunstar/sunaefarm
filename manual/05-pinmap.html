핀맵

Arduino Uno 전체 핀 배정 — 현장 적용 버전 (K-6060 / WO-100D 기준)

현장 적용 버전으로 업데이트됨
기존 28BYJ-48 스테퍼모터(D8~D11)와 12V 팬 구성에서 K-6060 DC 24V 모터 + WO-100D AC 220V 환풍기 구성으로 변경되었습니다.

변경 요약

❌ 제거된 핀 연결

  • D8~D11: ULN2003 IN1~4 (28BYJ-48 스테퍼)

✅ 새로운 핀 연결

  • D6: SSR 신호 → WO-100D 220V 환풍기
  • D7: DPDT 릴레이 A → K-6060 정회전(열기)
  • D8: DPDT 릴레이 B → K-6060 역회전(닫기)
  • D10: 리밋 스위치 A (열림 감지)
  • D11: 리밋 스위치 B (닫힘 감지)

전자밸브 핀 배정 (선택)

펌프 단독 대신 전자밸브 3개를 사용하는 경우 추가 릴레이 채널이 필요합니다.

핀연결비고
D524V 펌프 릴레이메인 펌프 ON/OFF
추가 릴레이 채널 1전자밸브 1 (민트)2nd 릴레이 모듈 필요
추가 릴레이 채널 2전자밸브 2 (레몬밤)
추가 릴레이 채널 3전자밸브 3 (바질)
Arduino Uno 디지털 핀 여유 확인
전자밸브 3개를 추가하면 디지털 핀이 부족할 수 있습니다. Arduino Mega 2560으로 업그레이드하거나, I2C 릴레이 익스팬더(PCF8574) 모듈을 사용하면 핀을 2개 추가 확장할 수 있습니다.

핀 배정 코드 선언 예시

// sunae_farm.ino — 핀 선언 (상단에 위치)

// ── 센서 핀 ─────────────────────────────
const int SOIL_PIN_1      = A0;  // 토양수분 민트
const int SOIL_PIN_2      = A1;  // 토양수분 레몬밤
const int SOIL_PIN_3      = A2;  // 토양수분 바질
const int DHT22_PIN       = 2;   // DHT22 DATA
const int BH1750_SDA      = 3;   // BH1750 소프트 I2C
const int BH1750_SCL      = 4;   // BH1750 소프트 I2C

// ── 액추에이터 핀 ────────────────────────
const int PUMP_RELAY      = 5;   // 24V 펌프 (LOW=ON)
const int FAN_SSR         = 6;   // 220V 환풍기 SSR (HIGH=ON)
const int MOTOR_OPEN      = 7;   // K-6060 정회전 (열기)
const int MOTOR_CLOSE     = 8;   // K-6060 역회전 (닫기)

// ── 리밋 스위치 ──────────────────────────
const int LIMIT_OPEN      = 10;  // 측창 열림 감지
const int LIMIT_CLOSE     = 11;  // 측창 닫힘 감지

// ── 모드 스위칭 ──────────────────────────
const int TOGGLE_AUTO     = 12;  // AUTO/MANUAL 토글
const int BTN_PUMP        = A3;  // 수동 펌프
const int BTN_FAN         = A4;  // 수동 팬
const int BTN_WINDOW      = A5;  // 수동 측창
