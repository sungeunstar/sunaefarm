프로젝트 개요

이 시스템이 무엇인지, 왜 만들었는지, 핵심 기능 요약

무엇을 만드는가

SUNAE FARM은 친구의 비닐하우스에 설치할 스마트팜 자동화 프로토타입입니다. 기존에는 사람이 직접 하루에 여러 번 화분 상태를 확인하고 물을 주고, 온도가 오르면 창문을 열고, 팬을 켜야 했습니다. 이 시스템은 그 반복 작업을 자동화하면서도, 필요할 때는 스마트폰이나 현장 버튼으로 직접 제어할 수 있게 설계되었습니다.

FARMOB이란?
FARM + TOMOB의 합성어입니다. SUNAE FARM 프로젝트의 시스템 코드명으로, 하드웨어·소프트웨어 시스템 전체를 가리킵니다.

설계 목표

1. 비닐하우스 환경에 최적화

비닐하우스는 여름에 내부 온도가 40°C를 넘기도 합니다. 방수가 되지 않는 장비는 습도와 물에 의해 고장납니다. 모든 전자 부품은 방수 인클로저에 수납하고, 센서는 내부식성 캐패시티브 방식을 사용합니다.

2. 인터넷 없이도 동작

농장은 WiFi가 불안정할 수 있습니다. WiFi가 끊어지더라도 Arduino 단독으로 기본 동작(관수·환풍·측창)을 유지할 수 있는 Fallback 로직을 설계합니다. 라즈베리파이는 네트워크 복구 후 자동으로 재연결됩니다.

3. 집에서 개발, 현장에서 설치

모든 코드는 GitHub으로 관리하고, 라즈베리파이에서 git pull 하면 최신 코드가 배포됩니다. 집에서 100% 동작 확인 후 하우스로 이동합니다. 현장 디버깅은 2배로 어렵습니다.

4. 3단계 모드로 유연한 제어

AUTO(자동)·REMOTE(원격)·MANUAL(현장 버튼) 3가지 모드를 물리 토글 스위치로 전환합니다. 어떤 상황에서도 사람이 개입할 수 있습니다.

하드웨어 구성 요약

두뇌 — 라즈베리파이 4B

  • Python 자동화 로직 실행
  • Flask 웹 대시보드 서버
  • 텔레그램 봇 알림
  • GitHub에서 코드 pull·배포
  • ngrok 외부 접속 터널

근육 — Arduino Mega 2560

  • 센서 직접 읽기 (아날로그·디지털)
  • 릴레이 ON/OFF 제어
  • DPDT 릴레이 제어 (K-6060 측창)
  • 물리 버튼 입력 처리
  • WiFi 없어도 독립 동작 가능

예상 총 비용

전체 부품 구매 예산은 약 ₩600,000 ~ ₩950,000 (30평 하우스 + 점적관수 포함)입니다. 자세한 항목은 구매 리스트에서 확인할 수 있습니다.
