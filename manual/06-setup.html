설치 방법

라즈베리파이 OS 설치부터 첫 실행까지 단계별 가이드

집에서 먼저 완전히 테스트하세요
현장(비닐하우스)에서 디버깅하는 것은 집에서보다 2배 이상 어렵습니다. 모든 기능이 집에서 동작 확인된 후에만 하우스로 이동합니다.

STEP 1 — 라즈베리파이 OS 설치

  1. Raspberry Pi Imager 다운로드
    공식 사이트(raspberrypi.com)에서 Raspberry Pi Imager를 PC에 설치합니다.
  2. OS 선택
    Raspberry Pi OS Lite (64-bit) 권장. 데스크탑 UI 불필요 — 용량과 메모리를 절약합니다.
  3. 고급 설정 (중요)
    Imager에서 ⚙️ 아이콘 클릭 후:
    • 호스트명: sunaefarm
    • SSH 활성화: ✅
    • WiFi 설정: SSID / 비밀번호 입력
    • 사용자 이름: pi / 비밀번호 설정
  4. MicroSD에 쓰기
    32GB 이상 MicroSD (Samsung U3 권장) 삽입 후 Write 클릭.
  5. 라즈베리파이 부팅
    SD카드 삽입 → 전원 연결. 최초 부팅은 2~3분 소요됩니다.

STEP 2 — SSH 접속

# 같은 WiFi 네트워크에서
ssh pi@sunaefarm.local

# IP 주소로 접속할 경우 (공유기 관리자 페이지에서 IP 확인)
ssh pi@192.168.x.x
처음 접속 시 fingerprint 경고가 나타납니다
Are you sure you want to continue connecting? (yes/no) → yes 입력

STEP 3 — 시스템 업데이트 및 의존성 설치

# 시스템 업데이트
sudo apt update && sudo apt upgrade -y

# Python 패키지 설치
pip3 install pyserial flask python-telegram-bot --break-system-packages

# arduino-cli 설치
curl -fsSL https://raw.githubusercontent.com/arduino/arduino-cli/master/install.sh | sh
sudo mv ~/bin/arduino-cli /usr/local/bin/

# arduino-cli 초기 설정
arduino-cli config init
arduino-cli core update-index
arduino-cli core install arduino:avr

STEP 4 — 코드 다운로드

cd ~
git clone https://github.com/sungeunstar/sunaefarm.git
cd sunaefarm

STEP 5 — Arduino 코드 업로드

# USB로 Arduino 연결 후 포트 확인
ls /dev/ttyUSB*
# 또는
ls /dev/ttyACM*

# 컴파일 + 업로드 (포트 번호를 확인한 것으로 대체)
arduino-cli compile --fqbn arduino:avr:uno arduino/
arduino-cli upload -p /dev/ttyUSB0 --fqbn arduino:avr:uno arduino/
포트가 보이지 않는 경우
Arduino를 USB로 연결하고 dmesg | tail -20을 실행하면 연결된 포트명을 확인할 수 있습니다.

STEP 6 — config.py 설정

nano raspberry/config.py

텔레그램 봇 TOKEN과 CHAT_ID를 입력합니다. 임계값은 초기값으로 시작하고 나중에 조정합니다.

STEP 7 — 첫 실행 및 테스트

# 자동화 메인 실행 (터미널에서 직접 테스트)
python3 raspberry/main.py

# 별도 터미널에서 웹 대시보드 실행
python3 raspberry/dashboard.py

브라우저에서 http://sunaefarm.local:5000 접속해 대시보드가 보이면 성공입니다.

STEP 8 — systemd 서비스 등록 (자동 시작)

sudo nano /etc/systemd/system/sunae-farm.service
[Unit]
Description=SUNAE FARM Automation
After=network.target

[Service]
ExecStart=/usr/bin/python3 /home/pi/sunaefarm/raspberry/main.py
WorkingDirectory=/home/pi/sunaefarm
User=pi
Restart=always
RestartSec=5

[Install]
WantedBy=multi-user.target
sudo systemctl enable sunae-farm
sudo systemctl start sunae-farm
sudo systemctl status sunae-farm  # 동작 확인
서비스 등록 후 자동 시작됩니다
라즈베리파이가 재부팅되거나 프로그램이 충돌해도 자동으로 재시작됩니다.
