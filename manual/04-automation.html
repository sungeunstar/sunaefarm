자동화 조건

관수·환풍·측창·긴급 알림 조건 및 임계값 설정 방법

임계값 설정 파일

모든 자동화 조건은 raspberry/config.py 한 파일에서만 수정합니다. 코드를 직접 수정할 필요 없이 이 파일의 숫자만 바꾸면 됩니다.

# raspberry/config.py

# ── 관수 설정 ────────────────────────────
SOIL_DRY_THRESHOLD   = 40    # 토양수분 40% 이하 → 관수 시작
VALVE_DURATION_SEC   = 60    # 전자밸브 열기 시간 (초) — 점적관수는 천천히 긴 시간 필요
WATERING_INTERVAL    = 1800  # 관수 후 대기 시간 (초, 30분)

# ── 환풍 설정 ────────────────────────────
FAN_TEMP_ON          = 28    # 온도 28°C 초과 → 팬 ON
FAN_TEMP_OFF         = 25    # 온도 25°C 이하 → 팬 OFF (히스테리시스)
FAN_HUM_ON           = 80    # 습도 80% 초과 → 팬 ON

# ── 측창 설정 ────────────────────────────
WINDOW_OPEN_TEMP     = 30    # 온도 30°C 초과 → 측창 열기
WINDOW_CLOSE_TEMP    = 25    # 온도 25°C 이하 → 측창 닫기

# ── 긴급 알림 ────────────────────────────
ALERT_TEMP           = 35    # 온도 35°C 초과 → 텔레그램 긴급 알림
ALERT_INTERVAL       = 300   # 알림 반복 간격 (초, 5분) — 스팸 방지

# ── 시리얼 포트 ──────────────────────────
SERIAL_PORT          = '/dev/ttyUSB0'
BAUD_RATE            = 9600

관수 (Watering)

토양수분 센서 3개(민트·레몬밤·바질)를 구역별로 독립 모니터링합니다. 조건이 충족된 구역의 전자밸브를 순차적으로 열어 점적관수합니다. PW-350SMA는 압력 스위치 내장으로 전자밸브가 열리면 자동 ON, 닫히면 자동 OFF됩니다. Arduino는 펌프를 직접 제어하지 않습니다.

1A0/A1/A2 → 토양수분 읽기 (0~100%)
240% 이하인 구역 확인
3관수 대기 시간(30분) 경과 여부 확인
4해당 구역 전자밸브 열기 → 수압 저하 → 펌프 자동 ON
560초 후 전자밸브 닫기 → 압력 회복 → 펌프 자동 OFF
6다음 구역으로 이동 (순차 처리) → 30분 타이머 리셋
전자밸브 동시 열기 금지
3개 구역 전자밸브를 동시에 열면 수압이 분산되어 점적기마다 유량이 불균일해지고 펌프 부하가 증가합니다. 반드시 순차 관수(구역 1 → 완료 → 구역 2 → 완료 → 구역 3)로 구현해야 합니다.
관수 중 측창 열기 금지
펌프가 동작 중일 때 측창이 열리면 바람에 의해 물이 전자 부품 쪽으로 튀어 고장날 수 있습니다. 소프트웨어에서 반드시 인터락(interlock)을 구현해야 합니다.

환풍 (Ventilation)

온도 OR 습도 조건이 충족되면 팬이 켜집니다. 단순히 ON/OFF가 아니라 히스테리시스(hysteresis) 방식으로 동작합니다.

히스테리시스란?

팬 ON 조건(28°C)과 팬 OFF 조건(25°C)을 다르게 설정하는 것입니다. 만약 같은 값이라면 온도가 28.0°C 근처에서 팬이 계속 켜졌다 꺼졌다 하는 채터링 현상이 발생합니다. 히스테리시스로 이를 방지합니다.

조건동작
온도 > 28°C팬 ON
습도 > 80%팬 ON
온도 ≤ 25°C AND 습도 ≤ 70%팬 OFF

측창 (Window)

K-6060 DC 24V 모터로 측창을 열고 닫습니다. D10/D11 리밋 스위치가 측창이 완전히 열리거나 닫혔을 때 모터를 자동 정지시킵니다. 스텝 수 제한 방식이 아니라 물리적 리밋 스위치로 끝 위치를 감지합니다.

측창 개방 중 관수 금지
측창이 열려 있는 상태에서 관수 명령이 들어오면 측창을 먼저 닫고 관수를 시작합니다. 순서: 측창 닫기 완료 확인 → 펌프 ON.

긴급 알림

온도가 35°C를 초과하면 텔레그램으로 즉시 알림을 발송합니다. 5분마다 반복 발송합니다 (스팸 방지를 위해 간격 설정).

# telegram_bot.py 발송 메시지 예시
⚠️ SUNAE FARM 긴급 알림
온도: 37.2°C (임계값 35°C 초과)
습도: 58%
시간: 2026-07-15 14:32:11
→ 즉시 확인이 필요합니다.
텔레그램 봇 설정 방법
  1. 텔레그램에서 @BotFather 검색
  2. /newbot 명령 → 봇 이름 설정
  3. 발급받은 TOKEN을 config.py에 입력
  4. 봇과 대화 후 CHAT_ID를 config.py에 입력
