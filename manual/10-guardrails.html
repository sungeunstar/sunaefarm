절대 금지 규칙

FARMOB 가드레일 — 코드·하드웨어·220V 안전 규칙 전체

220V 전기 안전 규칙 (최우선)

❌ 차단기 ON 상태에서 220V 배선 작업
220V 감전은 심실세동을 유발해 사망에 이를 수 있습니다. 배선 작업 전 반드시 하우스 분전반 차단기를 내리고 테스터기로 전압이 0인지 확인한 후 작업하세요.
❌ SSR 없이 220V 환풍기를 소형 릴레이로 직접 제어
구매 리스트의 5V 4채널 릴레이 모듈은 220V를 기술적으로 제어할 수 있지만, 접점 내구성이 산업 환경에 부적합합니다. 환풍기는 반드시 SSR 40A를 경유해 제어하세요.
❌ 맨 전선(피복 벗긴 상태) 터미널 연결
맨 전선을 나사 단자에 직접 연결하면 접촉 불량 및 단선이 발생합니다. 반드시 압착 단자(봉형 또는 핀형)로 처리 후 연결하세요.
❌ 접지(GND) 없는 220V 배선
장비 외함에 접지를 연결하지 않으면 누전 시 감전 사고가 납니다. 제어반 외함과 모든 220V 장비 외함에 접지선(녹색/노란색)을 연결하세요.

고전압·고전류 하드웨어 규칙

❌ DPDT 릴레이 A와 B 동시 ON
모터 열기(D7)와 닫기(D8) 릴레이가 동시에 켜지면 24V 전원이 쇼트됩니다. SMPS와 릴레이가 즉시 손상됩니다. 코드에서 반드시 한쪽을 OFF 확인 후 50ms 딜레이 이후 다른 쪽을 ON하세요.
❌ 리밋 스위치 없이 K-6060 모터 사용
측창이 완전히 열리거나 닫힌 후에도 모터가 계속 돌면 기어와 측창 프레임이 물리적으로 파손됩니다. D10/D11 리밋 스위치가 없으면 모터 구동 코드를 작성하지 마세요.
❌ SSR 방열판 없이 연속 동작
SSR 40A를 방열판 없이 사용하면 내부 온도가 급상승해 수 분 내에 손상됩니다. 방열판 + 써멀패드 설치를 확인한 후 SSR을 통전하세요.
❌ K-6060을 Arduino 핀에 직접 연결
Arduino 핀의 최대 출력은 40mA입니다. K-6060은 3.5A(=3500mA)를 필요로 합니다. 직접 연결 시 Arduino가 즉시 손상됩니다. 반드시 SMPS 24V + DPDT 릴레이 모듈을 경유하세요.

코드 규칙

❌ delay()로 센서 블로킹
delay()는 Arduino를 완전히 멈춥니다. 그 시간 동안 리밋 스위치·버튼·시리얼 수신이 모두 무시됩니다. 특히 모터 구동 중 리밋 스위치를 놓치면 기계 파손으로 이어집니다. millis() 기반 비블로킹 타이머로만 구현하세요.
❌ 센서 이상치(0 / -1 / NaN) 처리 없이 자동화 실행
DHT22가 -1을 돌려줄 때 "온도 -1°C → 히터 ON" 같은 오동작이 발생할 수 있습니다. 모든 센서 읽기 후 이상치 체크를 먼저 수행하고, 이상치 발생 시 해당 사이클을 건너뜁니다.
❌ 관수 중 측창 동시 작동
펌프·전자밸브 동작 중 측창이 열리면 물이 제어반 방향으로 튑니다. 소프트웨어 인터락으로 관수 중 측창 열기 명령을 반드시 차단하세요.
❌ WiFi 끊김 시 자동화 완전 정지
여름 하우스에서 WiFi 끊김 = 환풍 정지 = 작물 손상 가능. Arduino에 마지막 임계값을 저장해 라즈베리파이 없이도 기본 자동화가 유지되도록 Fallback 코드를 구현하세요.
❌ 핀 번호 하드코딩
코드 전체에 digitalWrite(7, LOW)처럼 숫자를 직접 쓰면 핀 변경 시 오류가 발생합니다. 파일 상단에 const int MOTOR_OPEN = 7;로 선언해 사용하세요.

센서 선택 규칙

❌ 저항식 토양수분 센서
전극 부식으로 2~4주 내 정확도 급락. 캐패시티브 v1.2 전용.
❌ DHT11 사용
정밀도 ±2°C로 28/30/35°C 임계값 구분 불가. DHT22 전용.
