원격 접속

SSH, ngrok 터널, 웹 대시보드 외부 접속 설정

접속 방법 비교

방법조건속도용도
로컬 SSH같은 WiFi빠름집에서 개발할 때
ngrok SSH 터널어디서든보통외부에서 코드 수정·디버깅
웹 대시보드 (로컬)같은 WiFi빠름센서값 확인·수동 제어
웹 대시보드 (ngrok)어디서든보통외부에서 모니터링·제어

로컬 SSH (집 내부)

ssh pi@sunaefarm.local
# 또는 IP 주소로
ssh pi@192.168.1.xxx

ngrok 설치 및 설정

  1. ngrok.com 가입 → 무료 계정 생성
  2. 라즈베리파이에 ngrok 설치 
    curl -sSL https://ngrok-agent.s3.amazonaws.com/ngrok.asc | sudo tee /etc/apt/trusted.gpg.d/ngrok.asc
echo "deb https://ngrok-agent.s3.amazonaws.com buster main" | sudo tee /etc/apt/sources.list.d/ngrok.list
sudo apt update && sudo apt install ngrok
  3. 인증 토큰 등록 (ngrok 대시보드에서 확인) 
    ngrok config add-authtoken YOUR_TOKEN_HERE

ngrok SSH 터널 실행

# 라즈베리파이에서 실행
ngrok tcp 22

# 출력 예시:
# Forwarding  tcp://0.tcp.ngrok.io:12345 -> localhost:22

# 외부에서 접속 (다른 PC / 스마트폰 터미널)
ssh pi@0.tcp.ngrok.io -p 12345
ngrok 무료 계정의 포트 번호는 매번 바뀝니다
ngrok을 재실행할 때마다 포트 번호가 달라집니다. 고정 도메인이 필요하면 ngrok 유료 플랜을 사용하거나, 텔레그램 봇으로 새 포트 번호를 알림받도록 코드를 추가할 수 있습니다.

ngrok 웹 대시보드 터널

# SSH 터널과 별개로 웹 대시보드용 터널 추가
ngrok http 5000

# 출력 예시:
# Forwarding  https://xxxx.ngrok-free.app -> http://localhost:5000

# 외부에서 브라우저로 접속
# https://xxxx.ngrok-free.app

ngrok 자동 시작 설정 (부팅 시)

sudo nano /etc/systemd/system/ngrok.service
[Unit]
Description=ngrok tunnel
After=network.target

[Service]
ExecStart=/usr/local/bin/ngrok tcp 22 --log=stdout
User=pi
Restart=always
RestartSec=5

[Install]
WantedBy=multi-user.target
sudo systemctl enable ngrok
sudo systemctl start ngrok
보안 주의
ngrok 터널을 열면 인터넷에서 라즈베리파이에 접속할 수 있게 됩니다. 반드시 강력한 SSH 비밀번호 또는 키 인증을 사용하세요. 기본 비밀번호 raspberry는 절대 그대로 두지 마세요.
