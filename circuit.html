Wokwi 임포트 방법
  1. wokwi.com 접속 → New Project → Arduino Uno
  2. 좌측 파일 목록에서 diagram.json 클릭
  3. 내용 전체를 다운로드한 파일로 교체
  4. 저장 후 시뮬레이션 실행

핀맵 (Arduino Uno)

센서

A0토양수분 센서 1민트
A1토양수분 센서 2레몬밤
A2토양수분 센서 3바질
D2DHT22 온습도10kΩ 풀업저항 필수
D3BH1750 SDA소프트웨어 I2C
D4BH1750 SCL소프트웨어 I2C

액추에이터

D5릴레이 CH1펌프
D6릴레이 CH2환풍팬
D8ULN2003 IN1측창 스테퍼
D9ULN2003 IN2측창 스테퍼
D10ULN2003 IN3측창 스테퍼
D11ULN2003 IN4측창 스테퍼

모드 스위칭

D12토글 스위치AUTO↔MANUAL
A3푸시버튼 1수동 제어
A4푸시버튼 2수동 제어
A5푸시버튼 3수동 제어

배선 주의사항

⚠️ DHT22: DATA 핀과 VCC 사이에 반드시 10kΩ 풀업저항 연결
⚠️ 릴레이 CH2 (팬): 12V 외부 전원 별도 연결 필요
⚠️ 관수 중 측창 열기 금지 — 물 튀김 사고 방지
⚠️ 스테퍼모터: 반드시 ULN2003 드라이버 경유. Arduino 직결 금지
⚠️ 모든 GND는 공통 접지 (Arduino GND)
