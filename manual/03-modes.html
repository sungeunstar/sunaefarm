동작 모드

AUTO / REMOTE / MANUAL 3단계 모드 전환 원리와 동작 방식

3가지 모드 개요

모드전환 방법누가 제어하는가라즈베리파이 역할
AUTO 토글 스위치 → AUTO 위치 센서값 기반 자동 판단 main.py가 조건 판단 후 명령 전송
REMOTE 웹 대시보드에서 명령 버튼 클릭 사람 (원격 수동) 대시보드 명령을 Arduino에 전달
MANUAL 토글 스위치 → MANUAL 위치 사람 (현장 버튼) 명령 무시됨 — Arduino가 독립 동작

AUTO 모드

가장 기본 모드입니다. 토글 스위치가 AUTO 위치에 있을 때 활성화됩니다. Arduino가 센서 데이터를 1초마다 라즈베리파이에 전송하고, main.py가 config.py의 임계값과 비교해 제어 명령을 내립니다.

1토글 스위치 D12 → HIGH 신호 (AUTO)
2Arduino → 센서값 시리얼 전송 (1초 주기)
3main.py → 임계값 비교 판단
4조건 충족 시 명령 전송 (PUMP:ON 등)
5Arduino → 릴레이·모터 제어 실행
안전 인터락
관수(펌프 ON) 중에는 측창 열기 명령이 들어와도 실행되지 않습니다. 물 튀김으로 인한 전자 부품 손상을 방지하는 SAFE-SEQUENCE 규칙입니다.

REMOTE 모드

웹 대시보드에서 버튼을 클릭하면 라즈베리파이가 Arduino에 즉시 명령을 보냅니다. 명령 실행 후 일정 시간이 지나면 자동으로 AUTO 모드로 복귀합니다.

REMOTE는 AUTO의 일시적 오버라이드입니다
토글 스위치는 AUTO 위치를 유지한 상태에서 웹으로 명령을 내리는 것입니다. 명령 후 AUTO 로직이 다시 작동합니다. MANUAL처럼 완전히 수동으로 넘어가는 것이 아닙니다.

접속 방법은 08. 원격 접속을 참고하세요.

MANUAL 모드

현장의 물리 버튼으로 직접 제어합니다. 토글 스위치를 MANUAL 위치로 내리면 Arduino가 라즈베리파이의 명령을 완전히 무시합니다. WiFi가 없거나 라즈베리파이에 문제가 생겼을 때 현장에서 직접 대응하는 비상 수단입니다.

현장 버튼 배정

버튼핀기능
버튼 1A3펌프 ON/OFF 토글
버튼 2A4환풍팬 ON/OFF 토글
버튼 3A5측창 열기/닫기 토글

LED 상태 표시

현재 모드를 LED로 확인할 수 있습니다.

LED 색의미
녹색 점등AUTO 모드 동작 중
청색 점등REMOTE 명령 실행 중
황색 점등MANUAL 모드 (라즈베리파이 명령 무시 중)
적색 점멸긴급 상황 (온도 35°C 초과)
