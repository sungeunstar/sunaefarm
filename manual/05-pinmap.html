핀맵

Arduino Uno 전체 핀 배정 — 현장 적용 버전 (K-6060 / WO-100D 기준)

Arduino Mega 2560 기준 핀맵
Arduino Uno는 디지털 핀이 부족해 전자밸브 3개 + SSR-2를 동시에 연결할 수 없습니다. Arduino Mega 2560을 사용하면 D22~D24 핀으로 전자밸브 3구역을 독립 제어할 수 있습니다. BH1750은 Mega의 하드웨어 I2C (D20/D21)를 사용합니다.

변경 요약

❌ 제거된 구성

  • 28BYJ-48 스테퍼모터 + ULN2003 드라이버
  • 12V DC 팬 + 12V 어댑터
  • 24V 다이어프램 수중 펌프
  • D3/D4 소프트웨어 I2C

✅ 현장 적용 구성

  • K-6060 DC 24V 모터 + DPDT 릴레이 (D7/D8)
  • WO-100D AC 220V 환풍기 + SSR-1 (D6)
  • 윌로 PW-350SMA + 전자밸브 × 3 (D22/D23/D24)
  • SSR-2 펌프 비상차단 (D5)
  • BH1750 하드웨어 I2C (D20/D21)
  • 리밋 스위치 A/B (D10/D11)

핀 배정 코드 선언 예시

// sunae_farm.ino — 핀 선언 (상단에 위치, Arduino Mega 2560 기준)

// ── 센서 핀 ─────────────────────────────
const int SOIL_PIN_1      = A0;  // 토양수분 민트
const int SOIL_PIN_2      = A1;  // 토양수분 레몬밤
const int SOIL_PIN_3      = A2;  // 토양수분 바질
const int DHT22_PIN       = 2;   // DHT22 DATA (10kΩ 풀업저항 필수)
// BH1750: Wire 라이브러리 사용 (D20/D21 자동 할당)

// ── 액추에이터 핀 ────────────────────────
const int PUMP_SSR2       = 5;   // 펌프 비상차단 SSR-2 (HIGH=차단)
const int FAN_SSR         = 6;   // 환풍기 SSR-1 (HIGH=ON)
const int MOTOR_OPEN      = 7;   // K-6060 정회전 (열기)
const int MOTOR_CLOSE     = 8;   // K-6060 역회전 (닫기)
const int VALVE_1         = 22;  // 전자밸브 민트 구역 (LOW=ON)
const int VALVE_2         = 23;  // 전자밸브 레몬밤 구역
const int VALVE_3         = 24;  // 전자밸브 바질 구역

// ── 리밋 스위치 ──────────────────────────
const int LIMIT_OPEN      = 10;  // 측창 열림 완료 감지
const int LIMIT_CLOSE     = 11;  // 측창 닫힘 완료 감지

// ── 모드 스위칭 ──────────────────────────
const int TOGGLE_AUTO     = 12;  // AUTO/MANUAL 토글
const int BTN_VALVE       = A3;  // 수동 관수
const int BTN_FAN         = A4;  // 수동 팬
const int BTN_WINDOW      = A5;  // 수동 측창

// ── 상태 표시 LED ─────────────────────────
const int LED_GREEN       = 9;   // 녹색 (AUTO 정상)
const int LED_YELLOW      = 13;  // 황색 (MANUAL/경고)
